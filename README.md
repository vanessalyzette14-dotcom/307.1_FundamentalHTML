# Intro to HTML

## Vocabulary:
    -   Elements - made up of HTML tags, and are a visual representation in the DOM
    -   Tags    - Begin and end an element '<> Content </>'
## Notes:
Parts of a webpage:
    -   HTML - Basic Structure / Layout
    -   CSS - Styling
    -   JS - Logic
Default Display Types:
    -   Display Block - Element will fill all horizontal space of container
